# Swift Vagrant

Simple playground for Swift lang.

## Quick Start

1. Setup environment
```
  git clone https://github.com/quamilek/swift-vagrant.git
  cd swift-vagrant
  vagrant up
  vagrant ssh
```
2. Run swift
```
$ swift
```
